Data Commons is intended to be a community project and seeks your involvement. Here are some ways you can help.


  • Publish data that can be included into Data Commons:
    • If you already have web pages that have the data in a human readable form, add schema.org markup to your page, that expresses the data you would like to publish. In addition to the terms in schema.org, you can also use the terms at schema.datacommons.org. To ensure that usage rights of the data is clear, make sure to use sdLicense on the page, to specify the license associated with data on the page.
    • Even if you don't already have web pages targeted at humans, you can publish the data using one of the formats listed here, using schemas from schema.org and/or schema.datacommons.org.
    • If you have a large dataset that might be of general interest and require help, please contact us. We might be able to help. We are in the process of building tools to help with conversion of large CSV files, etc. into formats that Data Commons can directly ingest.
    • Please get in touch with us if schema.org and schema.datacommons.org don't have the schemas you need for your data.
  • We need tools for visualization, summarization, answering queries, etc. Any help would be appreciated.
  • If you encounter issues using Data Commons, please get in touch with us! Please write to support@datacommons.org.
    • Please note that this time, we are only considering data that is licensed under CC BY.
